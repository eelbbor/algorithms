
Write a simple search engine which can locate a musical group from the attached dataset and return it's url fragment.  The url fragment is the ID column.  A stubbed out java object is provided - Searcher.java.

Guidelines
 - it's open book, OK to use books/internet as reference
 - use only core java, no libraries

Bonus points for any of the following:

 - memory efficient
 - scalable to larger datasets
 - data normalization
 - other cool features
